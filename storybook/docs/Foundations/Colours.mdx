# Colours

The Lyfeguard Design System uses a semantic colour system that adapts to light and dark themes.

## Theme-Aware Colours

All colours automatically adapt based on the active theme (light or dark). Use semantic colour tokens rather than hard-coding values.

## Usage

Always use CSS variables or the `vars` object from `globals.css`:

```tsx
import { vars } from '@lyfeguard/design-system/globals.css';

// ✅ Good
<div style={{ color: vars.color.theme.text.primary }}>

// ❌ Bad
<div style={{ color: '#000000' }}>
```

## Dark Mode

All colours automatically adapt in dark mode. The design system handles theme switching transparently - just ensure you're using theme-aware tokens.

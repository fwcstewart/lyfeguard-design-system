# Typography

The Lyfeguard Design System uses a consistent typography scale based on design tokens.

## Font Families

- **Sans-serif** - Primary font family for UI elements
- **Monospace** - For code and technical content

## Usage

Always use typography tokens:

```tsx
import { vars } from '@lyfeguard/design-system/globals.css';

// ✅ Good
<h1 style={{
  fontSize: vars.font.size.heading.lg,
  fontWeight: vars.font.weight.bold,
  lineHeight: vars.font.lineHeight.heading.lg,
  fontFamily: vars.font.sans
}}>

// ❌ Bad
<h1 style={{ fontSize: '24px', fontWeight: 'bold' }}>
```

## Best Practices

- Use heading sizes semantically (h1, h2, etc.)
- Maintain consistent spacing between text elements
- Ensure sufficient contrast for readability
- Use appropriate line heights for text blocks

# Welcome to Lyfeguard Design System

Welcome to the Lyfeguard Design System Storybook. This is your central hub for exploring our design tokens, components, and patterns.

## What's Inside

- **Foundations** - Core design principles including accessibility, colours, and typography
- **Components** - Reusable UI components with documentation and interactive examples
- **Layout** - Layout primitives and patterns for building consistent interfaces
- **Technical Patterns** - Advanced patterns and best practices for implementation

## Getting Started

Browse the sidebar to explore our design system. Each component includes:
- Overview and usage guidelines
- Interactive examples you can customize
- Props documentation
- Code examples

Use the theme toggle in the toolbar to switch between light and dark modes.
